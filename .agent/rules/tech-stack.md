# Tech Stack & Constraints

## Core Technologies
- **Frontend Framework**: Next.js 16 (App Router), React 19
- **Language**: TypeScript
- **Styling**: Tailwind CSS 3.4
- **Backend/Database**: Supabase (Postgres)
- **AI**: Google Vertex AI (Gemini)

## Approved Libraries (The White-List)
- **UI Components**: lucide-react, framer-motion, recharts
- **Utilities**: clsx, tailwind-merge, date-fns
- **Image**: sharp, heic2any, heic-convert
- **Email**: resend

## General Constraints
1. **Package Manager**: Use npm.
2. **Type Safety**: strict.
3. **Comments**: useful comments for complex logic.
