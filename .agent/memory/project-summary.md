# Project Summary

## Vision
A 'Design Studio' SaaS platform for visualizing handrail designs on customer photos using Generative AI (Google Vertex). It includes tenant management, admin dashboards, and lead capture.

## Core Features
1. **AI Visualizer**: Upload photo, apply handrail styles using Gemini AI.
2. **Tenant System**: Multi-tenant architecture with separate branding/dashboards.
3. **Admin Panel**: Manage tenants, stats, and configurations.
4. **Style Portfolio**: Manage handrail styles and assets.
5. **Lead Capture**: Collect user info before/after visualization.
